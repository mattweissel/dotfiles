Backups of configuration files for my two-monitor, makeshift home desktop environment.

## Defaults
- **Terminal**: Alacritty
- **Editor**: neovim
- **Browser**: Mozilla Firefox
- **Audio controller**: pulsemixer
- **Font**: AcPlus IBM VGA 8x16<br>(must be installed, see `fonts/`)

## Dependencies
- X
- pulseaudio
- dbus
- sudo

## To build and launch
```
$ <path>/xmonad/build && startx
```

## Supported applications
- Alacritty
- neovim
- Mozilla Firefox
- pulsemixer
- cmus
- htop
- Thunar
- Qalculate-gtk
- VLC Media Player
- OBS Studio
- Audacity
- Minecraft
- Steam<br>(must be pre-installed in chroot environment)
