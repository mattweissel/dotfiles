Backups of configuration files for my makeshift home desktop environment.

## Defaults
- **Terminal**: Alacritty
- **Editor**: neovim
- **Browser**: Mozilla Firefox
- **Audio controller**: pulsemixer

## Dependencies
- X
- pulseaudio
- dbus
- sudo

## Supported applications
- Alacritty
- neovim
- Mozilla Firefox
- pulsemixer
- htop
- Qalculate-gtk
- VLC Media Player
- OBS Studio
- Audacity
- Minecraft
- Steam (runs in chroot environment)
